# 💰 智能記帳系統使用指南

## 📌 概述

智能記帳系統是一個完整的個人記帳解決方案，可以幫助您輕鬆記錄、分析和管理日常支出。

**核心功能：**
- 📝 快速記錄支出（支持快速模板）
- 📊 自動統計分析（按分類、按時間）
- 📈 視覺化趨勢圖表
- ☁️ Google Sheets 同步
- 💾 本地和雲端雙重備份

---

## 🚀 快速開始

### 第一步：打開應用

訪問：`https://412078-bit.github.io/lareema2/expense-tracker.html`

### 第二步：記錄第一筆支出

1. **在「記錄支出」頁面**
   - 選擇日期（默認為今天）
   - 輸入金額
   - 輸入項目名稱（如：「午餐」）
   - 選擇分類（如：「飲食」）
   - 可選：添加備註
   - 點擊「新增記錄」

2. **或使用快速模板**
   - 點擊快速模板按鈕（如：「🥐 早餐」）
   - 自動填入分類和預設金額
   - 修改金額後點「新增記錄」

### 第三步：查看統計

1. **切換到「查看統計」頁面**
   - 自動顯示本月統計
   - 查看按分類的支出分佈
   - 查看日支出趨勢圖

### 第四步：導出和同步

1. **本地導出**
   - 在「詳細列表」頁面點擊「導出 CSV」
   - 下載到電腦

2. **Google Sheets 同步**
   - 進入「設定」頁面
   - 輸入 Google Apps Script URL
   - 點擊「同步到 Google Sheets」

## 📋 詳細功能說明

### 📝 記錄支出

#### 基本字段
- **日期**：支出發生的日期
- **金額**：支出的金額（單位：NT$）
- **項目名稱**：簡短描述（如：「星巴克咖啡」）
- **分類**：支出類型（8個預設分類）
- **備註**：可選的詳細說明

#### 支持的分類

| 分類 | 圖標 | 說明 |
|-----|------|-----|
| 飲食 | 🍔 | 餐飲、外食、飲料 |
| 交通 | 🚗 | 計程車、公車、停車費 |
| 購物 | 🛍️ | 日用品、衣服、配件 |
| 娛樂 | 🎮 | 電影、遊戲、休閒 |
| 生活 | 💡 | 電費、水費、房租 |
| 醫療 | ⚕️ | 掛號費、藥物 |
| 教育 | 📚 | 課程、書籍、培訓 |
| 其他 | 📌 | 其他支出 |

#### 快速模板

預先設置的常用項目，點擊可快速填充：
- 🥐 早餐（$50，飲食）
- 🍜 午餐（$80，飲食）
- 🍽️ 晚餐（$100，飲食）
- ☕ 飲料（$60，飲食）
- 🚕 計程車（$100，交通）
- 🚇 捷運（$50，交通）

#### 今日摘要

即時顯示：
- 📊 今日支出總額
- 📝 今日筆數
- 💰 本月支出總額
- 📌 本月筆數

### 📊 查看統計

#### 時間範圍選擇
- **本月**：當月 1 日至今
- **本週**：周一至今日
- **自訂**：選擇自己的日期範圍

#### 統計卡片
顯示四個核心指標：
1. **總支出**：所選時期的總金額
2. **平均支出**：每筆支出的平均金額
3. **最高支出**：最大的一筆支出
4. **筆數**：支出記錄總數

#### 按分類統計
- 柱狀圖展示各分類支出
- 進度條顯示佔比
- 百分比显示
- 點擊可查看詳細

#### 日支出趨勢
- 折線/柱狀圖展示每日支出
- 清晰呈現支出變化
- 便於發現規律

### 📋 詳細列表

#### 搜尋功能
- 輸入項目名稱或備註內容
- 實時模糊搜尋
- 支持多關鍵字

#### 篩選功能
- **按分類篩選**：只顯示特定分類的支出
- **按月份篩選**：選擇特定月份的記錄
- 支持組合篩選

#### 支出卡片
每張卡片顯示：
- 項目名稱和分類
- 支出日期和星期
- 支出金額
- 備註內容（如有）

#### 編輯和刪除
- 點擊支出卡片打開編輯窗口
- 可修改所有字段
- 可刪除記錄

#### 導出為 CSV
- 導出所有篩選後的記錄
- 可在 Excel/Google Sheets 中打開
- 便於進一步分析

### ⚙️ 設定

#### Google Sheets 同步

1. **輸入 GAS URL**
   - 從 [Google Apps Script 部署指南](./gs-deployment-guide.md) 獲取
   - 貼入 URL 並保存

2. **測試連接**
   - 點擊「測試連接」按鈕
   - 顯示連接狀態

3. **同步數據**
   - 返回主頁面
   - 點擊「同步到 Google Sheets」按鈕
   - 所有記錄將上傳到 Google Sheet

#### 數據管理

**導出備份（JSON）**
- 導出所有記錄為 JSON 文件
- 便於備份和轉移
- 可在任何設備中導入

**匯入備份**
- 從 JSON 文件恢復數據
- 支持合併操作

**清空所有記錄**
- 永久刪除所有記錄
- ⚠️ 此操作無法撤銷

## 💡 使用技巧

### 1. 日常記帳

**最佳實踐：**
- 每天睡前記錄一次
- 使用快速模板加速
- 備註欄記錄重要信息

**時間節省：**
- 使用快速模板省時 50%
- 複製相似記錄再修改
- 同步到 Google Sheets 一次性完成

### 2. 分析消費

**按分類分析：**
1. 進入「查看統計」
2. 查看「按分類統計」
3. 識別消費最多的分類

**按時間分析：**
1. 選擇不同時間範圍
2. 比較支出變化
3. 發現消費規律

### 3. 預算管理

**設置預算警告：**
- 觀察平均日支出
- 設定每日預算目標
- 定期檢查是否超支

**統計方法：**
1. 導出 CSV 到 Excel
2. 使用 Excel 函數進行進階分析
3. 製作自己的預算表

### 4. 資料同步

**何時同步？**
- 每週一次完整同步
- 或按需同步

**同步後的用途：**
- 在 Google Sheets 中做圖表分析
- 多設備查看
- 分享給家人/會計師

## 🔒 數據安全

### 本地存儲
- 數據保存在瀏覽器本地
- 不上傳到第三方服務器
- 同一瀏覽器可隨時訪問

### 備份建議
- 定期導出 JSON 備份
- 保存到雲端（Google Drive、Dropbox）
- 多設備不丟失數據

### 隱私保護
- 同步到 Google Sheets 前請檢查
- 只同步到自己的 Google 帳戶
- 不與他人分享 GAS URL

## 🆘 常見問題

### Q: 如何開始記帳？

A: 簡單三步：
1. 打開應用
2. 填寫支出信息
3. 點「新增記錄」

### Q: 支持多種貨幣嗎？

A: 目前只支持 NT$（新臺幣）。如需其他貨幣，可修改代碼。

### Q: 數據會丟失嗎？

A: 
- 不會自動丟失（存儲在瀏覽器）
- 清空瀏覽器緩存後會丟失
- 定期備份可保證安全

### Q: 可以在多個設備使用嗎？

A:
- 每個瀏覽器有獨立存儲
- 通過 Google Sheets 同步可跨設備
- 匯入 JSON 可轉移數據

### Q: 如何修改已記錄的支出？

A:
1. 進入「詳細列表」
2. 點擊要修改的支出卡片
3. 在彈出窗口中修改
4. 點「保存」

### Q: 支出可以批量導入嗎？

A:
- 支持 JSON 格式導入
- 不支持 CSV 直接導入
- 可手動轉換 CSV 為 JSON

### Q: Google Sheets 同步失敗怎麼辦？

A:
1. 檢查 GAS URL 是否正確
2. 點「測試連接」驗證
3. 檢查瀏覽器控制台錯誤
4. 參考 [Google Apps Script 部署指南](./gs-deployment-guide.md)

## 🎓 進階用法

### 自訂分類

編輯 `expense-tracker.js` 中的 CATEGORIES 對象：

```javascript
const CATEGORIES = {
    myCategory: { 
        name: '我的分類',
        icon: '🏠',
        color: '#ff6b6b'
    },
    // ... 更多分類
};
```

### 修改快速模板

編輯 `expense-tracker.js` 中的 QUICK_TEMPLATES：

```javascript
const QUICK_TEMPLATES = {
    myTemplate: { 
        project: '項目名稱',
        category: 'food',
        amount: 100 
    },
    // ... 更多模板
};
```

### 與 Google Sheets 進階整合

在 Google Sheet 中添加：
- 圖表和統計
- 月度預算表
- 年度總結報告
